• Pour inserer une vidéo, on doit :
    - aller dans Youtube (on peut aussi faire ca dans Tiktok),
    - choisir une vidéo,
    - cliquer sur "partager",
    - cliquer sur "intégrer",
    - coller le lien obtenu dans la page HTML !






  • Ou alors :
    - Vous télécharger une video sur votre ordinateur
    - Vous utiliser la basise "< video >"
    - Sans oublier la hauteur (height), la largeur (width), la source, et preciser la type (mp4)
